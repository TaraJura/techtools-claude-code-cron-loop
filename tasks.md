### TASK-089: Duplicate page detection and removal — find and remove identical or near-identical pages

Show a summary: "Found 5 duplicate groups (12 pages) — removing 7 pages would save ~3.2 MB". **Sensitivity control**: Provide a sensitivity slider (or dropdown: Exact Match Only / High Similarity / Medium Similarity / Low Similarity) that adjusts the Hamming distance threshold. These are common artifacts of scanning and batch imports. Show them in a separate "Blank Pages" section with the same review-and-remove workflow. **Integration**: Add a "Find Duplicates" button in the Pages tab/toolbar. Register with the command palette (TASK-048) as "Find Duplicate Pages", "Remove Blank Pages". Show results in a panel consistent with the existing UI style. Respect dark mode (TASK-029). Works with undo/redo (TASK-066) — page removal is an undoable operation.
